Canvas Game

This game was developed as part of project 3 of Udacity's Front End Web Developer Nanodegree. We were given a background of tiles and some specific rules, which we were allowed to build upon to create an online game similar to Frogger using HTML5 Canvas and Javascript. The game can be played online here.

Interactive Resume

This online interactive resume was developed as part of project 2 of Udacity's Front End Web Developer Nanodegree. We were given some existing javascript to modify and add content to. The resume can be found online here.

Online Portfolio

This online portfolio was developed as project 1 of Udacity's Front End Web Developer Nanodegree. We were given a static layout design mockup for how the portfolio should look and used the concepts learned in Udacity's Intro to HTML and CSS course to build the page.

Web Development Course Project - Wiki

I created a simple wiki for the final project of the Web Development course at Udacity. This wiki allows the user to browse content, look at content history as well as set up an account to create new and edit existing pages.

My First Rails App

Anyconf

AnyConf is a configuration library for python that wraps several other configuration libraries under a common API. I added YAML support to this library. This project as a whole was a collaborative effort.

About Me

I've always had a passion for solving problems and creating things for as long as I can remember. Software development was a natural fit for me and I've been successfully crafting software solutions for more than ten years now. Beyond continuously building my technical skills I also like to delve into the human factor involved in software development ranging from client interaction to team dynamics. I believe clear communication and mutual respect are key factors in determining project success.

When I'm not busy developing software you might find me playing with my daughter or working on a scene for one of my acting classes among a number of other things (because life is short, y'know; although there always seems to be time for video games and tv).

Thanks for stopping by and looking at some of my work.
